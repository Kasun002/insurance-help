# AI Insurance Help Center

## Project Brief

Build a modern customer support portal for a fictional insurance company using a modern UI framework (Next.js, React, or React Native).

The application should help users solve insurance-related questions through a combination of UI-based navigation and AI-assisted interaction.

Users should be able to:
- Browse information
- Search for answers
- Ask for help when needed

The experience should feel seamless between structured navigation and conversational assistance.

We want to evaluate your ability to build a practical, production-quality frontend application with meaningful AI integration.

---

## Objective

We are assessing your ability to:

- Build clean and scalable frontend systems  
- Design intuitive support experiences  
- Integrate AI into real user journeys in a useful way  
- Deliver a polished, working application

## Expected User Experience

The application should provide a seamless way for users to find insurance-related information and get help.

Users should be able to:
- Explore content easily  
- Ask questions in natural language  
- Receive meaningful guidance  

### Features
- Search insurance-related questions  
- Browse help topics  
- Ask questions via AI assistant  
- Receive step-by-step guidance  
- Ask follow-up questions  

### Sample Questions
- “How do I submit a car accident claim?”  
- “What does deductible mean in my policy?”  
- “Am I covered if my luggage is lost during travel?”  
- “How long does it take to process a claim?”  
- “What is the difference between term and whole life insurance?”
